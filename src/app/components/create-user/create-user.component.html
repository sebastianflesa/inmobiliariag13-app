Crear usuario

Registra una nueva cuenta utilizando el flujo oficial del Web App Cognito. Solo necesitas seguir los pasos y confirmar tu correo.

1 Haz clic en "Ir a registro" para abrir el Web App seguro de Cognito.
2 Completa tus datos personales y elige una contrasena.
3 Confirma el codigo enviado a tu correo electronico.
